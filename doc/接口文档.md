## 资金路由接口文档

#### 资金路由申请接口

请求参数描述

| 字段名               | 字段名                   | 类型   | 最大长度 | 是否必须 | 说明                                                         |
| :------------------- | :----------------------- | :----- | :------- | :------- | :----------------------------------------------------------- |
| routeNo              | 路由申请编号             | String | 64       | 是       | 路由申请编号全局唯一标识一次路由申请                         |
| applyNo              | 进件申请编号             | String | 64       | 是       | 资产方进件编号                                               |
| customName           | 申请客户姓名             | String | 100      | 是       |                                                              |
| customPhone          | 客户手机号码             | String | 20       | 是       |                                                              |
| customIdNo           | 客户身份证号码           | String | 32       | 是       |                                                              |
| customBalance        | 客户在贷余额             | String |          | 是       | 客户在贷余额包含零正整数单位是分                             |
| customLimit          | 客户在贷笔数             | String |          | 是       | 当前客户在系统中未完成的贷款笔数,只能是大于零正整数          |
| applyAmount          | 申请的金额               | String |          | 是       | 不包含零的正整数单位是分                                     |
| applyRate            | 申请的产品利率           | String |          | 是       | 0-1之间的小数最多保留六位                                    |
| applyAge             | 申请的年龄               | String |          | 是       | 不包含零的正整数                                             |
| limitStartDate       | 产品申请期限始           | String |          | 是       | 格式:yyyy-MM-dd                                              |
| limitEndDate         | 产品申请期限止           | String |          | 是       | 格式:yyyy-MM-dd                                              |
| applyProductTerm     | 产品期数                 | String |          | 是       | 不包含零的正整数                                             |
| applyRepayType       | 申请的还款方式           | String |          | 是       | 只能是[DEBX:等额本息,DEBJ:等额本金,YCXHBFX:一次性还本付息]其中一个 |
| microCreditSign      | 名企微贷标识             | String |          | 是       | 只能是[Y\|N]其中一个                                         |
| lineSign             | 线上/线下标识            | String |          | 是       | 只能是[ONLINE\|OFFLINE]其中一个                              |
| divisionCode         | 事业部编号               | String | 32       | 否       | 线下必须                                                     |
| divisionName         | 事业部名称               | String | 50       | 否       |                                                              |
| teamCode             | 事业部下的团队编号       | String | 32       | 否       | 线下必须                                                     |
| teamName             | 事业部下的团队名称       | String | 50       | 否       |                                                              |
| personalOrEnterprise | 个人/企业                | String |          | 是       | 只能是[PERSONAL\|ENTERPRISE]其中一个                         |
| uscCode              | 统一社会信用编码         | String | 32       | 否       | 企业必须填写                                                 |
| applyCrowd           | 申请人群                 | String |          | 是       | 只能是[A:本人,B:配偶,C:子女,D:股东,E:法人]其中一个           |
| idCardAddress        | 身份证地址               | String | 256      | 否       | 能取到就上送(暂时未确定是否参与路由规则)                     |
| homeAddress          | 住宅地址                 | String | 256      | 否       | 能取到就上送(暂时未确定是否参与路由规则)                     |
| businessAddress      | 营业执造地址             | String | 256      | 否       | 能取到就上送(暂时未确定是否参与路由规则)                     |
| manageAddress        | 实际经营地址             | String | 256      | 否       | 能取到就上送(暂时未确定是否参与路由规则)                     |
| companyAddress       | 公司地址                 | String | 256      | 否       | 能取到就上送(暂时未确定是否参与路由规则)                     |
| microOrConsume       | 小微/消费                | String |          | 是       | 只能是[MICRO\|CONSUME]其中一个                               |
| haveFileType         | 已收集到的文件资料的种类 | String |          | 是       | (A:身份证正面,B:身份证反面,C:营业执照,D:营业流水,E:收入证明,F:大头照)多个直接拼接例如:ABC |
